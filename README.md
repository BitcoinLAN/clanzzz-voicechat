# VoiceLink - Voice Chat App

End-to-end encrypted voice chat for gamers.

## Deploy to Render.com

### Step 1: Upload to GitHub
1. Create new repository on GitHub
2. Upload these files (drag and drop)
3. Click "Commit changes"

### Step 2: Deploy on Render
1. Go to https://render.com
2. Sign up with GitHub
3. Click "New +" â†’ "Web Service"
4. Connect your GitHub repository
5. Settings:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
6. Click "Create Web Service"

### Step 3: Done!
Wait 2-3 minutes and you'll get your URL.

## How to Use

Share your URL with friends. Everyone:
1. Enters their username
2. Enters the SAME room ID (like "game-room-1")
3. Clicks "Join Voice Chat"
4. Allows microphone access
5. Start talking!
